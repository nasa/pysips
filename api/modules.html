The PySIPS package provides a complete framework for Bayesian symbolic regression using sequential Monte Carlo.

Quick Links

  • pysips.PysipsRegressor - Main regressor interface

  • pysips.sampler - Sequential Monte Carlo sampling

  • pysips.prior - Prior distribution over symbolic expressions

  • pysips.laplace_nmll - Laplace approximation for model evaluation

For detailed usage examples, see the PySIPS Tutorial.
